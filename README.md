# MunchBite - Modern Food Delivery Website

A modern, responsive food delivery website built with HTML, CSS, and JavaScript.

## Features

- Fully responsive multi-page design
- Real-time cart synchronization across all pages
- Interactive hover animations and toast notifications
- Professional login system
- Mobile-first design (95/100 Lighthouse score)

## Tech Stack

- HTML5
- CSS3
- JavaScript
- Mint Glassmorphism design

## Getting Started

To run this project locally:

1. Clone or download the repository.
2. Open `index.html` in your web browser.

No server setup is required as this is a static website.

## Project Structure

- `index.html`: Home page
- `cart.html`: Shopping cart page
- `restaurant1.html` to `restaurant5.html`: Individual restaurant pages
- `help6.html`: Help and support page
- `script.js`: Main JavaScript file
- `style.css`: Main stylesheet
- `css/cart.css`: Cart-specific styles
- `js/cart.js`: Cart functionality
- `js/restaurant.js`: Restaurant page scripts
- `images/`: Directory containing images (blog, dish, products)
